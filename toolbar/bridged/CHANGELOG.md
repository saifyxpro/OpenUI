# @openui-xio/toolbar-bridged

## 0.0.1

### Initial Release

The OpenUI toolbar web app, served by the CLI proxy and rendered on top of the user's dev app.

- Element selection with visual highlighting
- Prompt input with element context
- Agent connection via WebSocket bridge
- Plugin system for framework-specific annotations
- Clipboard agent for copy/paste workflows
- Multi-chat support
- Draggable toolbar position
