- **Command Wrapping**: Run `openui -- npm run dev` to consolidate your dev setup into one command
- **Clipboard Agent**: Built-in clipboard-based agent for quick copy/paste workflows
- **Telemetry**: Opt-in pseudonymized usage tracking with three privacy levels (off, anonymous, full)
